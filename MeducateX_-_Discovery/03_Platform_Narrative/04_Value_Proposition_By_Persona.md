# 04 Value Proposition By Persona

**Purpose:** Map persona-specific value with grounded language.

**Last updated:** 2026-01-21

Surgeons: Structured review without directive prompts.
Residents: Curated educational pathways and transparent progress tracking.
Hospital Admins: Governance-ready dashboards and compliance reporting.
Compliance Teams: Audit trails and policy alignment checkpoints.

These values describe observed use cases rather than promises.
